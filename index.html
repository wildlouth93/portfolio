Projects

SnoopMoney

SnoopMoney is a robinhood clone. It's named after one of Robinhood's earliest investors, Calvin Broadus, Jr. This project allows users to track stocks and simulate stock purchases. SnoopMoney uses an external API to provide real-time stock information to users on the stocks currently held or watched by the user.

NoobTube

NoobTube is a Youtube clone with a twist. NoobTube is a video sharing platform that revolves around short video game clips posted by users. NoobTube is a niche social media application that allows users to interact with fellow gamers by posting videos and enjoying videos posted by other users.

What's in the Fridge?

What's in the Fridge? is an original application that uses an external recipe API to allow users to find meals. Users simply input ingredients into an ingredient to the user's ingredient list. From there, our app allows users to save recipes and populate a grocery list based upon saved recipes.

The Zone

The Zone is an original application that leverages a hidden ESPN API to allow users to track their favorite NFL teams and players. This application allows users to view team/player stats broken down by categories with user-friendly data visualizations.
